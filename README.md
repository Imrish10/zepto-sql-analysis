# Zepto Product Pricing & Inventory Analysis (SQL)

## Project Overview
This project analyzes Zepto product data to understand pricing strategies, discount patterns, and inventory availability.

The dataset contains over 3700 products across multiple categories.

## Database Design
The database was normalized into two relational tables:

- Categories
- Products

Primary and foreign keys were used to establish relationships between tables.

## Key SQL Concepts Used
- Joins
- Aggregations
- Window Functions
- Common Table Expressions (CTE)
- Data Cleaning
- Relational Schema Design

## Analysis Performed
- Category-wise product distribution
- Average discount by category
- Top expensive products
- Highest discount products
- Inventory availability
- Price per gram analysis
- Discount ranking using window functions

## Tools Used
SQL, MySQL Workbench

## Dataset Source
Kaggle - Zepto Product Dataset
